# TemperatureConverterPWA

This is a simple Progressive Web App (PWA) that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Files
- index.html - main page
- converter.css - styles
- converter.js - conversion logic and UI interactions
- manifest.json - PWA metadata
- sw.js - service worker for offline support
- icon512.png - app icon

## How to use
1. Open `index.html` in a browser.
2. To deploy: push to GitHub and enable GitHub Pages (main branch, /root).
3. Or upload files to PythonAnywhere (static HTML configuration).
